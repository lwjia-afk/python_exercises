# Mock coding session — timed

- **Time budget**: ~30 min

## Objective
Simulate the live coding portion. The point is the time pressure and verbalization, not novel problems.

## Task
Pick 2 problems you HAVEN'T re-done from the list:
- LC 49 (Group Anagrams)
- LC 167 (Two Sum II)
- LC 705 (Design HashSet)
- A SQL window-function problem

Set a 25-minute timer for each. THINK OUT LOUD as you go — pretend the interviewer is in the room. Record yourself if you can.

Afterwards, in `reflection.md` write:
- where did you get stuck?
- did you state assumptions before coding?
- did you discuss complexity?
- did you test on edge cases (`[]`, single-element, duplicates)?

## Expected output
2 timed sessions completed + reflection.
